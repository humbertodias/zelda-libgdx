# Games with libGDX

RPG game like Zelda using libGDX Library


## Prerequisites

1. Git 2.6+
2. Gradle 8.5+ (wrapper included)
3. Java 11+


## How to Play

Clone

```
git clone https://github.com/humbertodias/zelda-libgdx.git
```

Inside the folder

```
cd zelda-libgdx/src
```

Run

```
./gradlew desktop:run
```
